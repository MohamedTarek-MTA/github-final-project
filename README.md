# Simple Interest Calculator

This repository contains a bash script that calculates simple interest given the principal amount, annual rate of interest, and time period in years.

## Usage
To run the script, open your terminal, navigate to the directory containing the file, and execute the following command:
`./simple-interest.sh`

## Example
**Input:**
* Principal: 1000
* Rate of interest: 5
* Time period in years: 2

**Output:**
* The simple interest is: 100

## Explanation of Code
The script prompts the user to input three variables (`p` for principal, `r` for rate, `t` for time). It then calculates the interest using the formula `(p * t * r) / 100` and outputs the result to the console.
